# Impossible Coordination
What if strangers could work together toward common goals? We are going to see if we can make that happen with stigmergy. 
### Stigmergy
Stigmergy is an indirect coordination mechanism, by which traces in the environment (brake lights, stopsigns, etc.) stimulate the subsequent action (slow down, change lanes, etc.). Through stigmergy, coordination can occur without the need for centralized control, planning, or even communication. Stigmergy scales from the individual level to civilizational level complexities. Github is the closest thing to it right now, so lets make a Github for the rest of the world.
